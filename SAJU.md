# 사주(四柱) 완벽 가이드

## 📚 사주란 무엇인가?

사주(四柱)는 사람이 태어난 **년, 월, 일, 시**를 네 개의 기둥으로 표현한 동양 철학입니다.
각 기둥은 천간(하늘)과 지지(땅) 두 글자로 이루어져 총 8글자가 되므로 **사주팔자**라고도 합니다.

```
년주    월주    일주    시주
甲子    乙丑    丙寅    丁卯
↑       ↑       ↑       ↑
탄생    성장    나     미래
```

---

## 🎯 핵심 구성 요소

### 1. 사주 팔자 (saju_data)
**내용**: 태어난 년월일시를 천간지지로 표현
```json
{
  "year": "甲子",   // 년주: 조상, 어린 시절 (0-20세)
  "month": "乙丑",  // 월주: 부모, 청년기 (20-40세)
  "day": "丙寅",    // 일주: 본인, 중년기 (40-60세) ⭐
  "hour": "丁卯"    // 시주: 자녀, 노년기 (60세~)
}
```
**활용**: 
- 모든 사주 해석의 기본
- 일주(day)가 가장 중요 - 나 자신을 나타냄
- 일간(day의 첫 글자)이 모든 관계의 기준점

---

### 2. 오행 (elements)
**내용**: 목(木), 화(火), 토(土), 금(金), 수(水) 다섯 가지 기운의 분포
```json
{
  "wood": 4,   // 목: 성장, 인자함
  "fire": 2,   // 화: 열정, 예의
  "earth": 1,  // 토: 신뢰, 중용
  "metal": 0,  // 금: 결단, 의리
  "water": 1   // 수: 지혜, 유연함
}
```
**활용**:
- 부족한 오행 → 보완이 필요한 부분
- 과다한 오행 → 조절이 필요한 부분
- 균형 잡힌 오행 → 안정적인 삶

**해석 예시**:
- 목(4) 과다: 너무 착해서 손해 보기 쉬움
- 금(0) 부족: 결단력 부족, 우유부단

---

### 3. 십성 (ten_gods)
**내용**: 일간(나)을 기준으로 다른 천간/지지와의 관계
```json
{
  "stems": {      // 천간 십성
    "year": "편인",
    "month": "정인",
    "day": "비견",
    "hour": "겁재"
  },
  "branches": {   // 지지 십성
    "year": "정관",
    "month": "편재",
    "day": "편인",
    "hour": "정인"
  }
}
```

**십성의 의미**:
- **비견/겁재**: 형제, 동료, 경쟁자
- **식신/상관**: 표현력, 자녀, 재능
- **정재/편재**: 재물, 아버지, 경제력
- **정관/편관**: 직장, 명예, 남편(여성)
- **정인/편인**: 학업, 어머니, 지원자

**활용**: 성격, 적성, 대인관계 파악
- 정관이 많음 → 모범생 스타일
- 식신이 많음 → 예술적 재능
- 편재가 많음 → 사업가 기질

---

### 4. 대운 (dayun)
**내용**: 10년 단위로 바뀌는 인생의 큰 흐름
```json
{
  "start_age": 7,      // 7살부터 대운 시작
  "direction": "순행",  // 진행 방향
  "periods": [
    {"age": "7-16", "pillar": "丙寅"},   // 첫 대운
    {"age": "17-26", "pillar": "丁卯"},  // 둘째 대운
    {"age": "27-36", "pillar": "戊辰"}   // 셋째 대운
  ]
}
```

**활용**:
- 인생 계획 수립 (언제 무엇을 해야 할지)
- 좋은 시기/어려운 시기 예측
- 현재 35살 → 27-36 대운(戊辰) 확인

**해석 예시**:
- 대운이 용신과 맞으면 → 좋은 10년
- 대운이 기신이면 → 신중해야 할 10년

---

### 5. 신살 (sinsals)
**내용**: 태어날 때부터 가지고 있는 특별한 기운
```json
["역마살", "도화살", "화개살"]
```

**주요 신살**:
- **역마살**: 이동수, 변화가 많음, 해외운
- **도화살**: 이성운, 인기, 매력
- **화개살**: 예술성, 종교성, 고독
- **천을귀인**: 귀인의 도움
- **공망**: 허무함, 공허함

**활용**: 타고난 특성 파악
- 역마살 → 여행업, 무역업 적합
- 도화살 → 연예인, 서비스업 적합

---

### 6. 천을귀인 (helper_chars)
**내용**: 특정 천간/지지가 올 때 귀인운 발동
```json
["甲", "庚"]  // 이 천간이 오면 귀인의 도움
```

**활용**:
- 2024년 甲辰년 → 甲이 있으니 귀인의 해
- 대운이 庚午 → 庚이 있으니 귀인 대운
- 중요한 일은 귀인운이 올 때 추진

---

### 7. 공망 (empty_chars)
**내용**: 특정 지지가 올 때 공허한 운
```json
["戌", "亥"]  // 이 지지가 오면 헛수고
```

**활용**:
- 戌년이나 亥년은 조심
- 공망 대운에는 새로운 도전보다 수성
- 투자나 사업 시작 피하기

---

### 8. 지장간 (hidden_stems)
**내용**: 12지지 안에 숨어있는 천간 (고급 해석용)
```json
{
  "year": ["癸"],           // 子 속의 癸
  "month": ["己", "癸", "辛"],  // 丑 속의 己癸辛
  "day": ["甲", "丙", "戊"],    // 寅 속의 甲丙戊
  "hour": ["乙"]            // 卯 속의 乙
}
```

**활용**: 
- 정밀한 오행 분석
- 깊이 있는 성격 파악
- 일반인은 몰라도 됨 (전문가용)

---

### 9. 음양 균형 (yinyang)
**내용**: 양(陽)과 음(陰)의 비율
```json
{
  "yang": 5,  // 활동적, 외향적
  "yin": 3    // 수용적, 내향적
}
```

**활용**:
- 양 > 음: 외향적, 적극적, 리더십
- 음 > 양: 내향적, 신중함, 분석적
- 균형: 조화로운 성격

---

### 10. 용신 (yongshin) - 선택사항
**내용**: 사주의 균형을 맞춰주는 가장 필요한 오행
```json
"yongshin": "水"  // 물이 용신
```

**활용**:
- 용신 오행을 보강하면 운이 좋아짐
- 직업, 색깔, 방향 선택의 기준
- 水가 용신 → 북쪽, 검은색, 수산업 유리

---

### 11. 격국 (saju_type) - 선택사항
**내용**: 사주의 전체적인 품격과 유형
```json
"saju_type": "정관격"
```

**주요 격국**:
- **정관격**: 모범생형, 공무원, 안정 추구
- **편관격**: 도전자형, 군인/경찰, 추진력
- **정재격**: 실리형, 재물 추구, 사업가
- **편재격**: 투기형, 큰 돈, 위험 감수
- **식신격**: 예술가형, 자유로움, 창의성
- **상관격**: 비판형, 전문직, 완벽주의

---

## 💡 실제 활용 예시

### 1. 기본 성격 분석
```
오행: 목(4) 화(2) → 따뜻하고 인정 많음
십성: 정인 많음 → 공부 잘하고 어머니 영향 큼
신살: 도화살 → 이성에게 인기
```

### 2. 직업 선택
```
격국: 식신격 → 창의적 직업
용신: 火 → 요리, 디자인, 조명
역마살 → 해외 관련 업무
```

### 3. 연애/결혼 시기
```
도화살 + 정재운 → 결혼 적기
공망 대운 → 연애 신중하게
천을귀인 년도 → 좋은 인연
```

### 4. 투자/사업 시기
```
편재 대운 + 용신운 → 투자 적기
공망 → 투자 금지
정재운 → 안정적 수입
```

## 🎓 사주 읽는 순서

1. **일주 확인** → 기본 성격
2. **오행 분포** → 균형/불균형 
3. **십성 분포** → 재능과 적성
4. **대운 확인** → 현재와 미래
5. **신살 확인** → 특별한 기운
6. **용신 파악** → 보완점
7. **종합 해석** → 조언

---

## ⚠️ 주의사항

1. **사주는 가능성**입니다
   - 정해진 운명이 아닌 경향성
   - 노력으로 극복 가능

2. **균형이 중요**합니다
   - 좋은 것도 너무 많으면 독
   - 나쁜 것도 적당하면 약

3. **시대를 고려**하세요
   - 옛날 해석 그대로 적용 X
   - 현대적 재해석 필요

4. **전체를 봐야**합니다
   - 한 가지만으로 판단 금물
   - 종합적 해석 필요

## 📝 용어 정리

- **천간(天干)**: 10개의 하늘 기운 (甲乙丙丁戊己庚辛壬癸)
- **지지(地支)**: 12개의 땅 기운 (子丑寅卯辰巳午未申酉戌亥)
- **일간**: 일주의 천간, 나를 나타냄
- **용신**: 가장 필요한 오행
- **기신**: 피해야 할 오행
- **대운**: 10년 단위 운
- **세운**: 1년 단위 운
- **월운**: 1개월 단위 운
